What is DOM? What is the Purpose of DOM?

DOM is a short form of Document Object Model. It is used for the change or modify the web page structure style and content

How to select element from DOM?

We can select an element by multiple way form a DOM. If we want to select multiple element from dom, we can use document.getElementsByTagName() or document.getElementsByClassName() or document.querySelectorAll() method. If we want to select a single element of DOM we can use document.getElementById() or Document.querySelector() method
